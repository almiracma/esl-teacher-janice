=============================================
STUDENT PORTAL — How to Manage Your Content
=============================================

STUDENT PASSWORDS (Managed from Google Sheets!)
=================================================
Each student has their own personal password.
No more editing code — just add a row to your Google Sheet!

HOW TO ADD A NEW STUDENT:
1. Open your Google Sheet
2. Go to the "Students" tab
3. Add a row with these columns:

   Name | Password | Age Group | Total Lessons | Course | Status

   Example:
   Maria Santos | maria2026 | Adults | 8 | General English | Active

COLUMN GUIDE:
- Name:          Student's full name (must match their Bookings name)
- Password:      Any password you choose (e.g., firstname2026)
- Age Group:     Kids, Teens, or Adults
- Total Lessons: How many lessons they bought
- Course:        Which course they signed up for
- Status:        Active or Inactive (inactive students can't log in)

HOW TO CHANGE A PASSWORD:
Just edit the password in the Students sheet. That's it!

HOW TO DEACTIVATE A STUDENT:
Change their Status from "Active" to "Inactive".
They won't be able to log in anymore.


STUDENT DASHBOARD
==================
When students log in, they see a personal dashboard showing:
- Total lessons purchased
- Lessons completed
- Upcoming lessons
- Remaining lessons
- Full lesson history with dates and status

The dashboard reads from your Bookings sheet automatically.
For it to work correctly:
- The student's name in the Students sheet must MATCH
  their name in the Bookings sheet (same spelling!)
- Update the Status column in Bookings when lessons are done
  (change "Scheduled" to "Completed", "Cancelled", etc.)


HOW TO ADD A NEW TOPIC
=======================
Open the file: course-data.json

Find the right section (age group > course > level > topics)
and add a new entry using this template:

    {
      "title": "Your Topic Title Here",
      "description": "A short 1-2 sentence description of what students will learn.",
      "materials": []
    }

IMPORTANT: Make sure there's a comma after the } of the previous
topic before pasting a new one. Example:

    {
      "title": "Previous Topic",
      "description": "...",
      "materials": []
    },                              <-- comma here!
    {
      "title": "New Topic",
      "description": "...",
      "materials": []
    }


HOW TO ADD DOWNLOADABLE MATERIALS (PDFs)
=========================================
1. Put your PDF file in the materials/ folder:
   Example: materials/kids/general-english/beginner/my-worksheet.pdf

2. In course-data.json, update the topic's "materials" array:

    "materials": [
      {
        "label": "Worksheet - Greetings",
        "file": "materials/kids/general-english/beginner/my-worksheet.pdf"
      }
    ]

   You can add multiple files to one topic:

    "materials": [
      {
        "label": "Worksheet",
        "file": "materials/adults/business-english/beginner/worksheet.pdf"
      },
      {
        "label": "Vocabulary List",
        "file": "materials/adults/business-english/beginner/vocab.pdf"
      }
    ]


FILE NAMING TIPS
================
- Use lowercase letters
- Use hyphens instead of spaces: "my worksheet.pdf" → "my-worksheet.pdf"
- Keep names short and descriptive


HOW TO CHECK FOR ERRORS
========================
Before pushing changes, paste your course-data.json content into:
https://jsonlint.com
It will tell you if there are any formatting errors.


HOW TO PUSH CHANGES TO THE WEBSITE
====================================
After editing course-data.json (or adding PDFs):

1. Open Terminal
2. Navigate to your website folder:
   cd /Users/almira/esl_janice_website
3. Run these commands:
   git add .
   git commit -m "Updated course materials"
   git push

Your changes will be live in about 1-2 minutes!
